## Features

### Background Music
- Looped LoliRock theme music plays automatically.

### Navigation Tabs
- Each tab represents a section of the app with its own layout, content, and media:
  
#### The Band
- Introductory page with a quick overview of the series.
- Embedded audio, character images, and theme colors.

#### Iris, Talia, Auriana Tabs
- Each character has their own page:
  - Character bio
  - Unique background color
  - GIFs and themed styling

#### Songs
- A sortable, visual breakdown of songs from the show.
- Scatter charts show appearances and popularity.

#### Ratings
- IMDB-based episode ratings visualized with bar charts.

#### Playlists
- Visual breakdown of YouTube playlists by theme (Karaoke, Transformations, Behind-the-Scenes, etc.).
